## 操作

* 双击任何窗口tab可以触发全屏

* 部分窗口, 比如project窗口, 支持`speed search`: hover后直接输入需要搜索的词, 可以理解为屏幕显示文字搜索

* 可以把其他目录, 比如gopath下的其他项目, 加到本project, 通过目录右键 `Add directory to current project`

## 常用快捷键

* 全屏编码模式: `⇧ ⌘ F12`

* Project 窗口: `⌘ 1`

* Run 窗口: `⌘ 4`

* Terminal 窗口: `alt F12`

* 版本控制窗口: `⌘ 9`


## 不常用快捷键

* 搜索快捷键: `⇧ ⌘ A` 可能和其他软件截图快捷键冲突
